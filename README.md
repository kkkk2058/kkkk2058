## 🚀 Key Projects

### 🥽 AX-Log (LifeGo)
**1인칭 XR 기반 멀티모달 라이프로깅 시스템**
* **개요**: AR 글래스 및 HMD를 활용해 공간, 음성, 환경음, 이미지 등의 멀티모달 데이터를 수집 및 통합하는 시스템
* **주요 기술**: WebXR, SLAM3R, SpatialLM, Whisper, Qwen2.5-VL, Node.js, MariaDB
* **핵심 구현**: 임베딩 벡터 코사인 유사도 기반 의미적 이벤트 압축, 자연어 질의 기반 시맨틱 로그 검색 파이프라인 구축

### 💆‍♂️ 안면마비 자동 진단 기술 연구
**MediaPipe 및 AI 기반 안면 비대칭 자동 진단 시스템**
* **개요**: 표정 변화 구간을 정량적으로 분석하여 수동 평가의 주관성을 배제하고 원격 진단이 가능하도록 설계한 플랫폼
* **주요 기술**: MediaPipe, TabNet, WebXR
* **핵심 구현**: 얼굴 메쉬 좌표 기반 비대칭 피처 추출 고도화 및 하이퍼파라미터 튜닝을 통한 진단 정확도 향상(74% -> 84%)

### 🧊 Few-Shot Nerfacto
**밀도 그리드 통합과 희소 정규화를 활용한 3D 장면 복원**
* **개요**: 적은 수의 입력 이미지(Few-Shot)로 고품질 3D 장면을 복원하는 NeRF 기반 모델 연구
* **핵심 구현**: 희소 정규화(Sparse Regularization) 기법을 적용하여 불필요한 공간의 노이즈(Floating Artifacts)를 억제하고 표면 밀도 집중 유도
